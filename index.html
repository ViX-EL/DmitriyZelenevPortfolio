Выпускная квалификационная работа - дипломная работа: "Генерация трёхмерных пространств с помощью библиотеки Open GL"


Тестовый текст из диплома для примера.

Приложение предназначено для визуализации в трёхмерном пространстве процедурно генерируемых структур, написано на языке программирования Delphi и основано на обработчиках событий VCL Form и библиотеке OpenGL версии 1.

Разработанное конечное приложение для визуализации генерации трёхмерных пространств имеет 3 основные функции:

- Случайная генерация трёхмерного простого ландшафта с фиксированными размерами и наложением различных текстур в зависимости от высоты;

- Генерация структур на поверхности ландшафта в виде порталов ведущих в структуры комнат, находящиеся прямо под ними;

- Случайное размещение случайных картин с различными размерами и формами на стенах комнат.

В ходе выполнения выпускной квалификационной работы был разработан алгоритм процедурной генерации. На основе разработанных и исследованных алгоритмов спроектирована и реализована виртуальная картинная галерея с возможностью свободного перемещения в пространстве. Движение пользователя осуществляется с помощью стандартных устройств (клавиатура и мышь). Результатом ВКР является законченный программный продукт, представляющий графическое приложение визуализирующее картинную галерею. Картинная галерея представлена в виде группы помещений связанных между собой порталами.

GenEngineOpenGL3.3 - графический 2D/3D движок, основанный на графической библиотеке OpenGL версии 3.3 и выше, с инструментами процедурной генерации.


Тестовый текст для примера.

Приложение предназначено для графической визуализации (рендера) трёхмерных сцен различной сложности с применением современных техник визуализации и написано на языке программирования C++, вся визуализация основана на шейдерах с применением шейдерного языка программирования GLSL. Приложение разрабатывается в качестве будующей дипломной работы для института и находится в активной разработке, постоянно дорабатывается и дополняется.

На данном этапе реализованы следующие функции: - удобная камера для перемещения в 3D пространстве сцены; - загрузка текстур различных форматов, в том числе сжатых; - модель динамического освещения по Фонгу с тремя типами источников освещения: точечный, направленный и прожектор; - загрузка 3D моделей формата .obj; - свободное перемещение объектов сцены при помощи матричных преобразований; - рендер в текстуру; - загрузка и рендер скайбоксов; - отражения и преломления с применением техники environment mapping; - графический интерфейс пользователя для динамеческого добавления объектов на сцену и применению к ним трансформаций; - мультитекстурирование; - текстурные анимации.

Функции в разработке: - загрузка 3D моделей других форматов; - удобный менеджер объектов сцены; - удобный менеджер текстур и материалов; - генератор частиц; - доработка графического интерфейса пользователя; - инструменты процедурной генерации ланшафта и сохранения его в виде трёхмерных моделей; - инструменты процедурной генерации городов, пещер, подземелий.

В дальнейшем движок будет преобразован в набор инструментов для создания графических и игровых приложений, а так же для создания приложений дополненной реальности для настольных и мобильных платформ, в том числе ожидаемых устройств "очков дополненной реальности".
